# API Key 获取指南

获取API Key方式(免费赠送token可以直接试用)

* 阿里云百炼平台

## 阿里云百炼平台

网址:https://bailian.console.aliyun.com/cn-beijing/#/home

1.首页注册一下,或者直接通过支付宝、淘宝扫码登录注册即可

2.头部栏选择模型->模型用量->搜索deepseek-V3.2->开通(免费额度用完即停)

![](img/20260415112437_122_179.png)

3.创建API KEY

![](img/20260415113154_123_179.png)

4.把生成的API KEY复制粘贴到项目就可以使用
